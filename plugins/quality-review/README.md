# quality-review

Audit any artifact through 13 quality lenses. `/quality-review [level] [--fix] [<target>]`.

The rubric and its evidence live in the workspace note `notes/growth/quality-rubric.md`. The workflow skeleton follows the built-in /code-review (scope block, per-location verification, verdict ladder, sweep, synthesize-by-index); the reference extraction is in `notes/sources/code-review-workflow-reference.md`.

Domains ship as lens sheets under `skills/quality-review/references/`. Adding a domain means adding one sheet file.
